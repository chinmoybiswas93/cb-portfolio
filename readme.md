== Description ==

CB Portfolio is a comprehensive portfolio plugin that allows you to create a professional portfolio website with a modern two-column layout design. It features a complete admin dashboard for managing your portfolio data and a beautiful frontend display.

= Key Features =

* **Modern Two-Column Layout**: Fixed left sidebar with navigation and scrollable right content area
* **Complete Admin Dashboard**: Vue.js powered admin interface for easy portfolio management
* **Responsive Design**: Fully responsive design that adapts to mobile and tablet devices
* **Experience Management**: Add, edit, and organize your work experience with ordering capabilities
* **Project Showcase**: Manage your projects with descriptions, technologies, and links
* **Import/Export**: Backup and restore your portfolio data with JSON import/export
* **Mouse Spotlight Effect**: Interactive visual effect on desktop devices
* **Database Integration**: Custom database tables for efficient data storage
* **MVC Architecture**: Clean PHP code structure following WordPress coding standards
* **SEO Optimized**: Proper page titles, meta descriptions, and favicon support

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/cb-portfolio` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to 'CB Portfolio' in your WordPress admin menu to configure your portfolio.
4. Use the shortcode `[cb_portfolio]` to display your portfolio on any page or post.

== Frequently Asked Questions ==

= How do I display my portfolio on a page? =

After activating the plugin and adding your portfolio data through the admin dashboard, use the shortcode `[cb_portfolio]` on any page or post where you want to display your portfolio.

= Is the plugin responsive? =

Yes, the plugin is fully responsive and follows a mobile-first design approach. It adapts beautifully to all screen sizes from mobile phones to desktop computers.

= Can I import/export my portfolio data? =

Yes, the plugin includes a comprehensive Import/Export feature that allows you to backup your portfolio data as JSON and restore it when needed.

= How do I reorder my experience or projects? =

In the admin dashboard, each experience and project entry has up/down arrow buttons that allow you to reorder them. The changes are saved automatically.

= Does the plugin affect my site's performance? =

The plugin is optimized for performance with minified assets, efficient database queries, and minimal resource usage. The frontend is lightweight and fast-loading.

= Will the plugin use my site's title and favicon? =

Yes, the plugin automatically uses your WordPress site title, tagline, and favicon/site icon for proper branding and SEO.
